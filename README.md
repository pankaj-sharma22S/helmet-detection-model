# 🪖 Helmet Detection using CNN

A deep learning-based Helmet Detection System built with TensorFlow, Keras, OpenCV, and Scikit-learn. The model classifies images as **Helmet** or **No Helmet** using a Convolutional Neural Network (CNN).

## 🚀 Features
- Helmet vs No Helmet classification
- CNN model using TensorFlow/Keras
- Image preprocessing with OpenCV
- Dataset splitting and evaluation using Scikit-learn
- Model training and prediction

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Scikit-learn
- Matplotlib
